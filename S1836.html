Survey Rules

1️⃣ Daily Participation Limit

  • Maximum of 3 questions per day.
  • Survey stops automatically after reaching the limit.
  • Progress is saved using browser storage.

2️⃣ Time Limit

  • Each session has a 2-minute countdown.
  • If time expires, survey ends immediately.
  • User must return the next day to continue.

3️⃣ Answering Rules

  • Correct answer → + Ksh 10
  • Wrong answer → – Ksh 7
  • Wrong answer immediately stops the survey.

4️⃣ Withdrawal Rules

  • Minimum balance required: Ksh 100
  • Maximum withdrawal per request: Ksh 65
  • Withdrawal amount must not exceed current balance.

5️⃣ Account Activation

This is done prior to withdrawing then submit the withdrawal request Ksh 40.

6️⃣ Verification Process

  • User submits name and activation message.
  • Submission sent for admin verification.
  • Admin may approve or reject request.
  • Approved users' rewards will be credited to the submitted phone number.

7️⃣ Anti-Cheating Enforcement

If cheating is detected:

  • Survey terminates immediately.
  • Ksh 3 penalty is deducted.
  • User is blocked for the day.


Cheating Triggers:

  • Tab switching
  • Right-click usage
  • Copy, Paste, or Cut actions
  • Opening Developer Tools
  • Using restricted shortcuts (F12, Ctrl+Shift+I, Ctrl+U)
